Hey, the Codebase in the raw form has no database and hence has to be created.
Instead go to Pre-made Dataset and replace the Admin folder outside.
This would give you a dataset for 40 students of our class an also of 5 subjects to test.
===========================================================================================
Text File Format:
Teachers:
teadata.txt:	SubjectName FirstName Middle Last FilePath Password
Initial Password is Student Name
--------------------------------------------------------------------------------------
Students:
studdata.txt:	PRN First Middle LastName Password
Initial Password is PRN Number
---------------------------------------------------------------------------------------
In the Preset Dataset, First 10 PRNS have fathers' name as Password.
---------------------------------------------------------------------------------------
attendance sheet.txt
Date(YYYYMMDD) Attendence(on that date in form of strings. 1-Present 0-Absent)
---------------------------------------------------------------------------------------
password.txt
Admin Password
Initially the Password is "admin".
---------------------------------------------------------------------------------------
student_and_teacher_number.txt
Contains number of Students and Teachers respectively.

---------------------------------------------------------------------------------------
Each subject contains its own folder for attendance sheet.
===========================================================================================

Exception Handling is done on certain inputs such that invalid characters are not inputted.
